# Neo smartpen SDK Sample Page
Web SDK sample page using 'NeoSmart Pen'

## Installation 
``` sh
# Sample project setting
$ git clone https://github.com/NeoSmartpen/WEB-SDK-Sample
$ cd WEB-SDK-Sample
$ npm install
$ npm start
```
